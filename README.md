# Mental Health Awareness Website

A simple, responsive website designed to promote mental health awareness and provide supportive information in a calm and accessible way.

## Purpose

The purpose of this website is to:

-   Provide clear, easy-to-understand mental health information
-   Help users recognise common mental health challenges
-   Offer access to helpful external resources
-   Promote positivity through affirmations and supportive design

## Features

-   Hero section with welcoming message
-   Sticky navigation bar with logo
-   Mental health basics displayed in cards
-   Resources section with helpful links
-   Affirmation cards with imagery
-   Fully responsive layout using Bootstrap

## Built With

-   HTML5
-   CSS3
-   Bootstrap 5

## Design Approach

-   Calm, accessible colour palette
-   Rounded cards with subtle shadows
-   Simple typography for readability
-   Mobile-first responsive design

## Disclaimer

This website is for educational purposes only and is not a substitute for professional mental health advice or treatment.
